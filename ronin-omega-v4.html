📦 Empaquetar como APK
RONIN·Ω → Android App · 3 métodos
Método 1 — TWA (Trusted Web Activity) ⭐ Recomendado
1. Sube este HTML a GitHub Pages o Netlify (gratis). URL ejemplo: https://tu-user.github.io/ronin/
2. Instala bubblewrap: npm install -g @bubblewrap/cli
3. Inicializa: bubblewrap init --manifest https://tu-url/manifest.json

Añade este manifest.json junto al HTML: 
{
  "name": "RONIN·Ω MoE Research",
  "short_name": "RONIN·Ω",
  "start_url": "/",
  "display": "standalone",
  "orientation": "portrait",
  "background_color": "#070710",
  "theme_color": "#00ffa3",
  "icons": [
    {"src": "icon-192.png","sizes": "192x192","type": "image/png"},
    {"src": "icon-512.png","sizes": "512x512","type": "image/png"}
  ]
}
4. Build: bubblewrap build → genera app-release.apk
5. Firma el APK: bubblewrap build --skipPwaValidation y sube a Google Play o instala directamente.
Método 2 — Capacitor.js Offline total
1. npm init @capacitor/app ronin-apk · cd ronin-apk
2. Copia el ronin-omega.html como www/index.html
3. npx cap add android
4. npx cap sync · npx cap open android → Android Studio
5. En Android Studio: Build → Generate Signed Bundle/APK
✅ Sin servidor. Todo embebido en el APK. Funciona 100% offline. Acceso a cámara, micrófono, vibración nativos.
Método 3 — PWA + WebAPK (Chrome)
1. Abre la app en Chrome para Android
2. Menú → "Añadir a pantalla de inicio" → Chrome genera un WebAPK automáticamente
3. Para extraer el APK generado: adb shell pm path com.android.chrome.twa
⚡ Sin compilar nada. El APK se instala automáticamente. Ideal para testing rápido.
